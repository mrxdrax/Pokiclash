# 🎮 PokiClash

> A 2D Survivor-style Action Game built with **Unity** and **C#**.

PokiClash is an endless survival game where the player fights against continuous waves of enemies using multiple upgradable weapons such as Orbit Blades, Area Damage, and Projectile Bombs. The game features a power-up system that allows the player to become stronger over time and survive longer.

# 🕹 Weapons

## ⚔️ Orbit Blade

- Rotates around the player.
- Supports multiple blades.
- Upgradeable (Maximum 6 blades).
- Smooth hide/show animation.

---

## 💥 Bomb System

- Automatically targets the nearest enemy.
- Projectile-style movement.
- Explosion damage with area effect.
- Upgradeable bomb count.

---

## 🛡 Area Weapon

- Expands from the player.
- Smooth scale animation.
- Deals continuous damage.
- Upgradeable attack radius.
